# MauiCollectionViewEstudo

Este projeto é uma aplicação experimental desenvolvida com **.NET MAUI** (.NET 10), criada especificamente para o estudo e exploração do componente **CollectionView**.

## 🚀 Objetivo

O objetivo principal deste repositório é servir como um ambiente de aprendizado para:
- Implementação de listas dinâmicas com `CollectionView`.
- Customização de `ItemTemplate`.
- Uso de diferentes layouts (Linear, Grid).
- Gerenciamento de seleção e estados visuais.
- Performance e virtualização de listas no MAUI.

## 🛠️ Tecnologias Utilizadas

- **Framework:** .NET MAUI (.NET 10.0)
- **Linguagem:** C# / XAML
- **Plataformas Suportadas:**
  - Android (API 21+)
  - iOS (15.0+)
  - MacCatalyst (15.0+)
  - Windows (WinUI 3)

## 📁 Estrutura do Projeto

- **MauiCollectionViewEstudo/**: Contém o código-fonte principal, incluindo XAMLs e lógica de negócio.
  - **Platforms/**: Códigos específicos para cada sistema operacional.
  - **Resources/**: Ativos do aplicativo (ícones, imagens, fontes e estilos).
  - **MainPage.xaml**: Página inicial (atualmente com o template padrão de contador).

## 🏃 Como Executar

1. Certifique-se de ter o **Visual Studio 2022** (ou VS Code com extensões MAUI) e o **Workload .NET MAUI** instalados.
2. Clone este repositório.
3. Abra o arquivo de solução `.slnx` ou a pasta do projeto.
4. Selecione a plataforma de destino (Android, iOS ou Windows).
5. Compile e execute (F5).
